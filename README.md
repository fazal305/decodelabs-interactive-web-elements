# JS Interactive Lab

DecodeLabs Frontend Development Internship — Batch 2026  
Project 3: Interactive Web Elements

## Project Goal

Add interactivity to a webpage using JavaScript, DOM manipulation, buttons, toggles, and dynamic content updates.

## Interactive Elements

- Dark / Light Mode Toggle
- Dynamic Counter
- JavaScript FAQ Accordion
- Live Character Counter
- Colour Palette Generator
- Show / Hide Content Cards

## DecodeLabs Engineering Standards Applied

- Used `const` for DOM references
- Used `let` for changing state
- No `var`
- Used `addEventListener()`
- No inline `onclick`
- Used `textContent` instead of `innerHTML`
- Used `js-` prefix for JavaScript hooks
- Used `is-` prefix for visual state classes
- Followed Input → Process → Output loop
- CSS handles visuals, JavaScript handles behaviour

## Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript
