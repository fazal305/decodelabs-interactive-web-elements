## Project Overview

JS Interactive Lab is a modern interactive web application built using HTML5, CSS3, and Vanilla JavaScript. The project demonstrates core frontend development concepts including DOM manipulation, event handling, dynamic content updates, localStorage persistence, and responsive design.

This project was developed as part of the DecodeLabs Frontend Development Internship Batch 2026.

## Features

### 🌙 Dark / Light Mode Toggle

* Theme switching using CSS variables
* localStorage persistence
* Dynamic button text updates

### 🔢 Dynamic Counter

* Increment
* Decrement
* Reset
* Positive and negative state indicators

### ❓ JavaScript FAQ Accordion

* Expand/collapse functionality
* Only one item open at a time
* Smooth CSS transitions

### ✍️ Live Character Counter

* Real-time character tracking
* Warning states near limit
* Maximum character enforcement

### 🎨 Colour Palette Generator

* Generates random colour palettes
* Dynamic DOM creation using createElement()
* Click-to-copy hex codes using Clipboard API

### 👁️ Show / Hide Content Cards

* Toggle content visibility
* Dynamic button text updates
* Smooth expand/collapse transitions

## DecodeLabs Engineering Standards Applied

* const and let only
* No var usage
* addEventListener() for all events
* No inline onclick handlers
* textContent for safe DOM updates
* js-* JavaScript hook classes
* is-* visual state classes
* Input → Process → Output workflow
* Separation of structure, style, and behaviour

## Technologies Used

* HTML5
* CSS3
* Vanilla JavaScript
* Local Storage API
* Clipboard API
